### What I learned

Creating a landing page using css grid and flexbox.

```css
.hero-section {
  max-width: 125rem;
  margin: 0 auto;
  text-align: left;

  display: grid;
  grid-template-columns: 1fr 1fr;
}

.hero-img {
  display: flex;
  flex-direction: column;
  justify-content: center;
}

.hero-text {
  padding: 0 4.5rem;
  display: flex;
  flex-direction: column;
  justify-content: center;
}
```
